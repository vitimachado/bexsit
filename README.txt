# react.js e node.js Project (Mmartan)

Hosting Firebase: https://get-servicos.web.app/

FRONT (REACT.JS)
BACK (NODE.JS)

Obs: foi usado um código para fazer um web scraping e popular um banco no firestore. (webscraping.js)

--- FRONT (REACT.JS)

Primeiro você deve ter instalado o node e o npm.

1. Instalar dependências
    `npm install`.

2. Iniciar server  
    `npm start`

3. Abra browser: [http://localhost:3000](http://localhost:3000)

---

--- BACK (NODE.JS)

Primeiro você deve ter instalado o node e o npm. Abra a pasta "./api".

1. Instalar dependências
    `npm install`.

2. Iniciar server  
    `node index.js`

---

--- webscraping (NODE.JS)

Primeiro você deve ter instalado o node e o npm. Abra a pasta "./api".

1. Instalar dependências
    `npm install`.

2. Iniciar server  
    `node webscraping.js`
